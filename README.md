## 系统行为
1. 用户发送指令让服务器执行
2. 用户发送指令让服务器转交给设备执行
3. 用户发送数据让服务器转交给设备存储

protoc --go_out=./ .\message.proto
protoc --go_out=./ .\register.proto
protoc --go_out=./ .\profile.proto
protoc --go-grpc_out=. .\register.proto

protoc --go-grpc_out=. .\register.proto
protoc --go-grpc_out=. .\profile.proto
